Book Explorer is a responsive web application built using HTML and Tailwind CSS
that allows users to discover popular books in a clean and modern interface. 
The platform features a structured navigation bar (Home, Favorites, About), 
a hero section with a call-to-action, and a responsive book grid displaying 
book covers, titles, and authors. The project demonstrates frontend development
skills including layout design, responsive grids, reusable components, and clean 
UI structure. It is designed as part of a lab focused on DOM interactivity,
JavaScript modules, and localStorage integration for managing favorite books.
